# ⚙️ Setup do arsenal — DutyX

O DutyX funciona **sem instalar nada** (modo básico). Esta pasta é pra quem quer o **modo turbo** — conectar as ferramentas que dão o melhor resultado. Vá com calma: liga uma de cada vez, quando precisar. (A skill **`/configurar`** te conduz por aqui.)

## Básico vs Turbo

| | Básico (padrão) | Turbo (arsenal ligado) |
|---|---|---|
| Transcrever vídeo | você cola a legenda/fala | **Whisper local** transcreve sozinho |
| Métricas do Instagram | você informa os números | **Windsor** puxa tudo automático |
| Anúncios | você descreve a campanha | **Meta** traz os dados reais |
| Banco de campeões | fica no próprio sistema (`saidas/`) | espelhado no **Notion** (opcional) |

**Comece no básico. Ligue o turbo quando quiser.** Nada disso bloqueia o uso do DutyX.

## Guias
- `whisper-local.md` — transcrição no seu PC (de graça)
- `windsor.md` — métricas do Instagram
- `meta-ads.md` — anúncios
- `notion.md` — banco de campeões (opcional)
- `mcp.template.json` — modelo pra MCP local/customizado

## Regra de ouro
Toda ferramenta externa é um lugar a mais onde dá pra travar. Por isso: se algo não conectar, **não trave** — volta pro modo básico e segue. O DutyX foi feito pra você nunca ficar preso.
