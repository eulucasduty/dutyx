DutyX
Guia completo · como funciona e como usar

O DutyX é um sistema que você instala numa pasta e usa com o Claude Code. Você conta o seu negócio pra ele uma vez, e a partir daí ele te ensina e constrói junto cada frente: site, landing page, tráfego, conteúdo, métricas, vendas. A IA é seu funcionário 24h; o DutyX diz a ela como trabalhar no seu negócio, do seu jeito.

A regra que torna o DutyX único: o QUÊ antes do COMO.
O mercado só executa. O DutyX ensina e depois executa — antes de construir, ele te explica o conceito, confirma que você entendeu, e só então constrói. É isso que te torna dono, não refém. Por isso ele não deixa você pular essa parte.

01Como começar

Uma vez na vida:

  1. Instalar o Claude Code — conta Claude + Node + npm install -g @anthropic-ai/claude-code. Passo a passo no README.md.
  2. Abrir a pasta do DutyX no Claude Code (digita claude).
  3. Digitar /instalar → entrevista rápida (5 min) → o sistema te conhece.

Depois disso, todo dia que for trabalhar: /abrir — ele te dá o "bom dia" e diz o que fazer.

04O arsenal (turbo — tudo grátis)

O DutyX funciona sem nada (modo básico). Pra ter o melhor resultado, ligue o arsenal com /configurar — montado pra rodar só com ferramentas gratuitas:

  • Conectores nativos do Claude (Conectores → Novo conector → login): Windsor (métricas, grátis) e Meta (anúncios).
  • Whisper local (transcrição) — você baixa 1x no PC, grátis. É o que faz o /aprender-video e o /metricas transcreverem.
  • Imagem real: Pollinations (sem cadastro) ou Gemini free.
  • Site de agência: 21st.dev (copiar componente = grátis) + Framer Motion + skill frontend-design.
  • Pago = só opcional: Higgsfield (imagem premium), 21st.dev Magic. Nunca necessário.

06O fluxo do dia a dia

  1. /abrir → ele te diz onde parou e o que fazer.
  2. Chama a frente que você quer (/site-foda, /carrossel, /vendas…).
  3. Ele ensina → confirma → constrói → aprende seu gosto.
  4. /salvar no fim.

Achou uma técnica nova num reel? /aprender-video e o sistema incorpora.

07As regras de ouro (a alma do DutyX)

  • O quê antes do como — você sempre entende antes de receber.
  • Ninguém trava — se faltar uma ferramenta, ele faz no modo básico; nunca te deixa preso num erro.
  • Free-first — o melhor resultado é de graça; pago é só extra.
  • Na sua marca — tudo sai com a sua cara, não genérico.
  • Build-in-public — é o sistema que o Lucas usa de verdade, crescendo à vista.
