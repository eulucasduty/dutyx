# Bloco — esqueleto de uma LP que converte

> Modelo de seções pronto pra preencher. Brand-neutral: as cores/fontes vêm de `_contexto/marca.md`. O motor `claude-design` transforma isso em HTML/CSS de alta fidelidade.

## Seções (na ordem)

1. **HERÓI** (primeira dobra)
   - Promessa (headline): o resultado que a pessoa ganha, claro em 3s.
   - Subtítulo: pra quem é + como entrega.
   - CTA primário (botão): verbo de ação ("Quero começar", "Garantir minha vaga").
   - (opcional) imagem/mockup do produto ou um elemento visual forte.

2. **DOR** — 1 bloco curto que nomeia o problema que a pessoa vive.

3. **VIRADA / SOLUÇÃO** — o que você oferece + mecanismo em 3 passos.

4. **PROVA** — depoimentos / prints / números / antes-e-depois.

5. **O QUE TEM DENTRO** — lista do que está incluído (bullets curtos, com benefício, não só feature).

6. **OFERTA + PREÇO** — ancoragem ("de R$X por R$Y") + garantia + CTA.

7. **FAQ** — 3 a 5 objeções respondidas.

8. **CTA FINAL** — última chamada + escassez (só se for real).

## Regras de ouro do bloco
- CTA repetido em pelo menos 3 pontos da página (topo, meio, fim).
- Máximo 2 fontes; corpo ≥ 16px; bom contraste.
- Parágrafos curtos. Bullet em vez de muro de texto.
- Mobile primeiro: a maior parte vai abrir no celular.
