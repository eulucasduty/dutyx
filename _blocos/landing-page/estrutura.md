# Bloco — esqueleto de uma LP que converte

> Modelo de seções pronto pra preencher. Brand-neutral: as cores/fontes vêm de `_contexto/marca.md`. O motor `claude-design` transforma isso em HTML/CSS de alta fidelidade. Copy pronta de exemplo (headlines + seção a seção preenchida): `copy-exemplos.md`.

## Seções (na ordem)

1. **HERÓI** (primeira dobra)
   - Promessa (headline): o resultado que a pessoa ganha, claro em 3s.
   - Subtítulo: pra quem é + como entrega.
   - CTA primário (botão): verbo de ação ("Quero começar", "Garantir minha vaga").
   - (opcional) imagem/mockup do produto ou um elemento visual forte.

2. **DOR** — 1 bloco curto que nomeia o problema que a pessoa vive.

3. **VIRADA / SOLUÇÃO** — o que você oferece + mecanismo em 3 passos.

4. **PROVA** — depoimentos / prints / números / antes-e-depois.

5. **O QUE TEM DENTRO** — lista do que está incluído (bullets curtos, com benefício, não só feature).

6. **OFERTA + PREÇO** — ancoragem ("de R$X por R$Y") + garantia + CTA.

7. **FAQ** — 3 a 5 objeções respondidas.

8. **CTA FINAL** — última chamada + escassez (só se for real).

## Variante: LP de captura (quando o objetivo é LEAD, não venda)
Curta de propósito — quem roda tráfego pra lead usa essa: **herói** (promessa + o que a pessoa recebe em troca do contato) → **1 prova** (a mais forte que tiver) → **formulário nome + WhatsApp** (ou e-mail, se a entrega é por e-mail) → **tela/página de obrigado** (onde a conversão é medida). Sem dor longa, sem FAQ — quanto menos rolagem até o formulário, melhor.

## Regras de ouro do bloco
- **LP não tem menu de navegação** — nenhuma rota de fuga além do CTA (logo sem link, sem "Home/Sobre").
- CTA repetido em pelo menos 3 pontos da página (topo, meio, fim).
- Máximo 2 fontes; corpo ≥ 16px; bom contraste.
- Parágrafos curtos. Bullet em vez de muro de texto.
- Mobile primeiro: a maior parte vai abrir no celular.
- **Formulário mínimo:** nome + WhatsApp e mais nada (ou botão direto pro `wa.me`). Cada campo a mais derruba conversão.
- **Urgência só se for verdade** (vagas reais, data real). Contador fake queima a página inteira.
- Objeção não mora só no FAQ: a garantia responde o medo, a prova responde a desconfiança, o mecanismo responde o "será que funciona pra mim".
